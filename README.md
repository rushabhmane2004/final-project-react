# Paradise Nursery Shopping Application

## Project Name: Paradise Nursery

Paradise Nursery is an online plant shop built with React and Redux. Users can browse a variety of houseplants, add them to a shopping cart, and manage their cart items dynamically.

## Features
- Browse plants grouped into categories
- Add plants to cart with live cart count updates
- Increase, decrease, or remove items from cart
- View total cost per item and overall cart total
- Checkout button with "Coming Soon" message
- Continue Shopping button to return to product listing

## Tech Stack
- React (Vite)
- Redux Toolkit
- CSS

## Getting Started
```bash
npm install
npm run dev
```
